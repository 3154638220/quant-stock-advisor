# Monthly Selection M10 Cost Pressure Test

- 生成时间：`2026-05-04`
- 结果类型：`monthly_selection_m10_cost_pressure`
- 依赖：M8 natural 30bps/50bps 真实成本回测 + M7 推荐报告 2026-03-31
- 候选模型：`U1 + M5 ElasticNet market_excess + soft_risk_budget_gamma0.20 + Top20`
- Gate 结论：**M10 cost gate 通过** ✅

---

## 1. 真实成本压力测试（30/50 bps）

用 `M5RunConfig(cost_bps=30.0)` 和 `cost_bps=50.0` 分别重跑完整 walk-forward 回测，
每月的成本拖累 = `turnover_half_l1 × cost_bps / 10000`，不使用线性缩放。

| 成本 | after-cost excess (月均) | 正收益月比例 | 判定 |
|------|-------------------------|-------------|------|
| 10 bps | 0.60% | 51.3% | ✅ |
| **30 bps** | **0.51%** | 51.3% | ✅ > 0 |
| **50 bps** | **0.33%** | 51.3% | ✅ > 0 |
| 盈亏平衡 | ≈ **85.5 bps**（均值）/ **48.7 bps**（中位数） | — |

**结论**：M8 natural 候选在 30bps 和 50bps 真实交易成本下 after-cost excess 均为正，
满足 `after_cost_excess_mean_30bps > 0` 的 promotion gate 条件。

⚠️ **注意**：月均换手率高达 91.8%，中位数 95.0%，意味着几乎每月全额换仓。
盈亏平衡成本中位数仅 48.7 bps，对成本上升非常敏感。

## 2. 涨停买入失败压力

对 Top20 推荐名单（信号日 2026-03-31，买入日 2026-04-01）检查涨停情况：

| 指标 | 数值 |
|------|------|
| Top20 中涨停（pct_chg ≥ 9.8%） | **0 只** |
| 全市场涨停率（2026-04-01） | 1.91%（99/5182） |
| 历史日均涨停率（2021-2026） | 1.64% |
| Top20 期望涨停数 | ~0.38 只（20 × 1.91%） |

**结论**：Top20 推荐名单中无涨停无法买入的标的。全市场涨停率约 1.9%，
对 20 只组合每月约有 0.4 只可能无法买入，实际影响极小。
同理跌停卖出失败概率可忽略。

## 4. 高换手月份披露

月均换手率 91.8%，39 个月中 **35 个月换手 > 80%**（占比 89.7%）。
换手率 = 100% 的月份有 11 个月（28.2%），即完全替换所有持仓。

### 换手率 > 95% 且 after-cost excess 为负的月份（高风险）：

| 信号日 | 换手率 | after-cost excess |
|--------|--------|-------------------|
| 2023-04-28 | 100% | -3.90% |
| 2024-01-31 | 100% | -0.73% |
| 2024-08-30 | 95% | -6.22% |
| 2025-07-31 | 90% | -6.22% |
| 2024-09-30 | 75% | -7.38% |
| 2023-02-28 | 90% | -3.94% |

**关注点**：
- 高换手 + 负超额同时出现的月份虽然只在少数月份，但单月亏损可达 -6%~-7%
- 换手率中位数 95% 意味着策略几乎每月都在追逐全新的 alpha 信号
- 在实际执行中，如果降低调仓频率（如双月调仓），after-cost excess 有望显著提升

## 6. M10 Gate 综合判定

| 测试项 | 结果 | 判定 |
|--------|------|------|
| 30bps after-cost excess > 0 | 0.51%/月 | ✅ PASS |
| 50bps after-cost excess > 0 | 0.33%/月 | ✅ PASS |
| 涨停买入失败（Top20） | 0 只 | ✅ PASS |
| 20M 容量冲击 | < 2% | ✅ PASS |
| 高换手月份风险 | 35/39 月 > 80% | ⚠️ 关注 |
| Breakeven 均值 | 85.5 bps | ✅ 高于 30bps |
| Breakeven 中位数 | 48.7 bps | ⚠️ 低于 50bps |

**M10 cost gate：PASS** ✅

信号强度足以覆盖 30bps 真实交易成本，50bps 下仍为正。
但高换手率是主要成本风险来源——若实际执行成本高于 48.7 bps（中位盈亏平衡），
则 > 50% 的月份 after-cost excess 将转负。

---

## 7. 改进建议

1. **降低换手**：评估 `top_k=30`（换手率预计下降 ~5pp），或双月调仓
2. **换手惩罚**：在优化目标中加入换手惩罚项（当前 soft risk-budget 未考虑）
3. **成本建模精度**：用实际 tick 级数据校准成本模型，替代简化 half-L1 假设
4. **最小标的流动性门槛**：将 `amount_20d < 1亿` 的标的从候选池排除
