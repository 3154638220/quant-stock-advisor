# 文档目录

长期维护的说明文档保留在 `docs/` 根目录；按日期生成的研究报告归档到 `docs/reports/<YYYY-MM>/`。

- `项目算法建模详解.md`：算法、统计口径与模型说明。
- `月度选股流程说明.md`：月频选股主线、候选池、脚本/模块边界、产物与 promotion 纪律。
- `backtest_report.md`：当前回测报告入口，配套数据为 `backtest_report_data.json`。
- `plan.md`：当前研究计划。
- `研究脚本契约化与配置治理.md`：研究 manifest、配置治理和脚本模块化治理记录。
- `reports/`：历史研究报告、诊断报告和证据链归档。

新报告若是一次性实验产物，优先放入 `docs/reports/<YYYY-MM>/`；只有会被 README 或月度主线持续引用的文档才留在根目录。
