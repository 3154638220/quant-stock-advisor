### 4.0 当前回测口径说明（重要）

- 本报告数值来自 `**config.yaml.backtest`**：`composite_extended` 线性打分 + Top-10 + `**risk_parity**` + **月末调仓** + **预过滤** + **换手约束** + `**tplus1_open`** +涨停 `**redistribute**`。
- **未使用** `sort_by = xgboost` 或 `tree_score` 排序。
- 与历史文档中 `**close_to_close` + 等权** 的正收益表 **不是同一实验**；对比前请先统一执行模式与组合权重方法。

### 4.1 XGBoost 模型 Rank IC 为负（历史记录）

查看 `data/models/xgboost_panel_ee2108f515be/bundle.json`（若工件仍存在）：

```json
"metrics": {
    "train_rank_ic": -0.16114580398302147,
    "val_rank_ic": -0.07358242336614988
}
```

**核心结论（适用于 xgboost 排序链路）**：

- Rank IC 为负时，按 `tree_score` 降序选 Top-K 会系统性偏向未来表现更差的股票。
