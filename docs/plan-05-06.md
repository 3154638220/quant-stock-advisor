### 4.2 2026-05-06 实际验证结果 ❌ FAIL Gate

对 M5 concept delta（`m5_concept_delta_2026-05-06_monthly_long.csv`）中 ExtraTrees + concept 与 M8 hard-cap baseline（`m8_concentration_regime_2026-04-30_monthly_long.csv`）进行配对比较（152 个匹配月份，2023-01 至 2026-02）：

| 指标 | ExtraTrees+Concept | HardCap Baseline | Delta |
|------|-------------------|------------------|-------|
| 月均超额（gross） | 2.01% | 1.94% | **+0.08%** |
| 月均超额（after-cost） | 1.92% | 1.84% | +0.08% |
| 月正收益率 | 63.2% | 68.4% | −5.2pp |
| 配对 t 检验 | — | — | t=0.168, **p=0.87** |

年度切片 delta：
| 年份 | Delta (gross) | 结果 |
|------|--------------|------|
| 2023 | +0.23% | ✓ |
| 2024 | +1.45% | ✓ |
| 2025 | **−1.66%** | ✗ |
| 2026 (2m) | +1.27% | ✓ |

**Gate 判定**：
- ❌ 月均 delta > 0.1%：+0.08%，未达标
- ❌ 配对检验 p < 0.1：p=0.87，远未达标
- ✓ 年度切片：3/4 年正（允许一年接近零）

**结论**：ExtraTrees+concept 在 walk-forward 下对 hard-cap baseline 的增量不显著（delta≈0.08%，p=0.87）。M5 leaderboard 排名第一主要来自模型方差而非概念因子的 stable alpha。暂不纳入 promotion 候选，概念因子方向转由 M13-B（个股绑定）验证。

## 5. M13-B：P5-V2 概念/主题 Alpha（个股绑定）

**背景**：P5-V1 使用板块等权 breadth，因为 AkShare 概念板块成分历史变化不完整，个股-概念绑定缺失。V1 delta≈0 的根本原因是 ElasticNet 无法捕获非线性的主题扩散信号，且个股绑定不精确。

### 5.1 数据工程目标

**当前状态（2026-05-06）**：`a_share_concept_membership` 表已建（schema: symbol, concept_code, snapshot_date）但为**空表**。EM（东方财富）API 已被封（`stock_board_concept_cons_em` 连接拒绝），同花顺 API 无成分股映射端点，Futu API 不可用。M13-B 核心数据工程受阻。

**已知尝试**：
| API | 状态 |
|-----|------|
| `ak.stock_board_concept_cons_em()` | ❌ 连接拒绝（EM 被封）|
| `ak.stock_concept_cons_futu()` | ❌ 不可用 |
| `ak.stock_board_concept_info_ths()` | ✓ 仅概念级元数据，无成分股列表 |

构建 PIT-safe 的个股-概念绑定表：

```python
# 目标 schema
# symbol | concept_id | concept_name | entry_date | exit_date (nullable)
# 仅记录截止信号日仍在板块内的标的
```

数据源优先级：
1. AkShare `ak.stock_board_concept_cons_em()` — 当前成分（仅最新快照）
2. 手工记录近 2 年历史变化（若 AkShare 无历史）
3. 保守处理：若无 exit_date 记录，仅使用近 6 个月以内的成分记录

### 5.2 候选因子（V2）

```python
features_concept_v2 = [
    "concept_member_count",          # 个股所属概念板块数（广度代理）
    "hot_concept_membership",        # 是否属于当月热门板块 Top-10（等权成员涨幅排序）
    "concept_ew_return_1m",          # 所属最大板块近 1 月等权收益
    "concept_max_return_1m",         # 所属概念中近 1 月最高涨幅板块的收益
    "concept_inflow_breadth",        # 所属概念板块中净流入成员占比
    "concept_return_dispersion",     # 板块内收益离散度（高离散度 = 主题扩散初期）
]
```

### 5.3 PIT 验证要求

| 检查项 | 要求 |
|--------|------|
| 成分记录时间 | 早于信号日 entry_date |
| 退出记录处理 | 若无 exit_date 记录，仅使用 rolling 60 天内的快照 |
| 板块收益计算 | 以信号日 t 的成员计算 t-1M 到 t 的收益，不含 t+1 信息 |
| 覆盖率要求 | U1 候选池覆盖率 ≥ 60% 才进入 M5 主线 |

### 5.4 进入 M5 的 Gate

## 7. M14：龙虎榜因子接入

**Alpha 假设**：机构席位持续买入（连续出现于龙虎榜）= 中长期资金认可，可能有月度延续性。个人投机席位游资特征 = 短期动量噪声，可作逆向信号。

### 7.1 因子设计

```python
features_lvl5 = [
    "lhb_institutional_net_buy_1m",   # 近 1 月机构席位净买入金额
    "lhb_appearance_count_1m",        # 近 1 月出现在龙虎榜次数
    "lhb_buy_sell_ratio_1m",          # 近 1 月机构净买入 / 总成交额
    "lhb_is_top_buyer_1m",            # 近 1 月前 5 大买入席位含机构 flag
]
```

### 7.2 数据源

AkShare `ak.stock_lhb_stock_statistic_em()` — 按票聚合历史龙虎榜记录。

PIT 要求：龙虎榜为 T 日公告，当日信号即可使用（无延迟）。

### 7.4 2026-05-07 实际研究结果

**数据源变更**：因 EM API 被封，改用新浪财经 `stock_lhb_detail_daily_sina`。该 API 返回每日 LHB 明细（股票代码、收盘价、涨跌幅、成交量/额、上榜原因），但**无机构/游资席位区分**。直接 HTTP 拉取（绕过 AkShare 缓存），覆盖 2021-01-04 ~ 2026-04-30（1289 个交易日，94,314 行，5,543 只独特股票）。

**M1 质量诊断**：
- 月覆盖 U1 候选池 10–17%（远超 5% 门槛），每月 500–800 只股票上榜
- 数据为 T 日收盘后披露 → PIT-safe

**因子设计（因无机构席位数据，调整因子列表）**：
| 因子 | 说明 |
|------|------|
| `feature_lhb_appearance_count_1m` | 近月上榜次数 |
| `feature_lhb_appearance_count_3m` | 近季上榜次数 |
| `feature_lhb_recent_5d` | 近 5 日是否上榜 (0/1) |
| `feature_lhb_avg_change_1m` | 上榜日平均涨跌幅 |
| `feature_lhb_avg_amount_1m` | 上榜日平均成交额 |
| `feature_lhb_is_bullish_1m` | 上榜原因含看涨信号 |
| `feature_lhb_is_bearish_1m` | 上榜原因含看跌信号 |
| `feature_lhb_is_high_turnover_1m` | 因高换手上榜 |

**M5 delta 结果（39 个月 walk-forward）**：

| 模型 | after-cost 超额 | vs PV ExtraTrees delta |
|------|----------------|------------------------|
| LHB + ExtraTrees | **1.95%** | **+0.70%** |
| LHB + ElasticNet | 1.04% | −0.60% |
| LHB + XGBoost | 0.44% | −0.32% |
| PV ExtraTrees (baseline) | 1.24% | — |
| PV ElasticNet (best PV) | 1.64% | — |

**vs HardCap Baseline（M8 indcap3）**：
| 指标 | LHB+ExtraTrees | HardCap Baseline | Delta |
|------|---------------|------------------|-------|
| 月均 after-cost | 1.95% | 2.24% | **−0.19%** |
| 2023 | 1.78% | 2.37% | −0.58% |
| 2024 | 1.24% | 1.00% | +0.24% |
| 2025 | 2.97% | 3.41% | −0.44% |

**Gate 判定**：
- ✅ M1 覆盖率 10–17%（≥ 5%）
- ✅ LHB 对 ExtraTrees 有显著增量（+0.70% vs PV ExtraTrees）
- ❌ 未超过 hard-cap baseline（−0.19%）
- ❌ LHB 改善局限于树模型；ElasticNet/XGBoost 均为负 delta

**结论**：LHB 因子对树模型（特别是 ExtraTrees）有边际价值，可作为**辅助特征族**在 ensemble 中使用，但不足以独立成为 promotion 候选。因无机构席位数据，无法构建原始设计的机构净买入/买卖比因子。建议保持 LHB 数据持续更新（每月增量），在后续 ensemble research 中作为低权重辅助因子。

## 9. M16：结构化公告事件因子（长期）

**优先级低，但 Alpha 潜力高**。业绩预告、股票回购、大股东减持、解禁公告等对月度收益有显著预测性。

### 9.1 事件类别

| 事件 | 预期方向 | 数据源 |
|------|----------|--------|
| 业绩预告（大幅上调） | 正向 | AkShare `ak.stock_zygc_ym()` |
| 回购公告（金额 / 市值比高） | 正向 | AkShare `ak.stock_repurchase()` |
| 大股东减持计划 | 负向 | AkShare `ak.stock_share_hold_change()` |
| 解禁（近 1 月大量解禁） | 负向 | AkShare `ak.stock_circulate()` |
| 可转债转股压力（临近强赎） | 负向 | 需额外数据源 |

### 9.2 PIT 挑战

公告事件必须区分：**公告日（announcement_date）** vs **事件生效日**。所有事件特征必须以公告日作为信号可用日，不得使用事件效果日。

建议在 `src/features/` 下新建 `event_factors.py`，并强制要求每个事件数据源通过 `src/data_fetcher/migrations.py` 的 PIT 校验流程。
