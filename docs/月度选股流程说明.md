# 月度选股流程说明

本文档描述当前仓库保留的唯一主线：月频选股。旧的逐日输出、解释报告、事后评估与事件关注度扫描已经从主线删除。

## 1. 主线边界

| 项目 | 当前口径 |
| --- | --- |
| 频率 | 月频，信号日为每月最后一个交易日 |
| 目标 | 从可交易股票池中选出未来一个月更可能跑赢市场的 Top-K |
| 当前生产默认 | `monthly_selection_u1_top20_indcap3_hardcap_baseline` |
| 默认候选池 | `U1_liquid_tradable` |
| 默认持仓数 | Top20 |
| 行业约束 | 单行业名称最多 3 只 |
| 交易时点 | 月末卖出当月 Top20，下一交易日开盘买入下一月 Top20 |

生产准入边界只看 `configs/promoted/promoted_registry.json`。未登记的研究候选可以保留证据链，但不能写回 `config.yaml.example` 或本地生产 `config.yaml` 的默认主线。

## 2. 数据准备

月度链路仍依赖日线表，但日线只是月频特征、标签和可交易性判断的输入，不再触发逐日名单输出。

常用入口：

```bash
python scripts/fetch_only.py --max-symbols 50
python scripts/build_industry_map.py
python scripts/fetch_fundamental.py --max-symbols 100
python scripts/fetch_fund_flow.py --max-symbols 100
python scripts/fetch_shareholder.py --latest-n 4
```

核心数据表：

| 表 | 作用 |
| --- | --- |
| `a_share_daily` | OHLCV、成交额、换手率、月末信号日与 open-to-open 标签 |
| `a_share_fundamental` | PIT 基本面估值与质量因子 |
| `a_share_fund_flow` | 资金流因子 |
| `a_share_shareholder` | 股东户数与变化率因子 |
| `data/cache/industry_map.csv` | 行业名称与行业约束 |

## 3. 月度研究链路

按当前 M2 到 M8 的研究口径，主线脚本如下：

```bash
python scripts/run_monthly_selection_dataset.py --config config.yaml.backtest
python scripts/run_monthly_selection_oracle.py --config config.yaml.backtest
python scripts/run_monthly_selection_baselines.py --config config.yaml.backtest
python scripts/run_monthly_selection_multisource.py --config config.yaml.backtest
python scripts/run_monthly_selection_ltr.py --config config.yaml.backtest
python scripts/run_monthly_selection_report.py --config config.yaml.backtest
python scripts/run_monthly_selection_concentration_regime.py --config config.yaml.backtest
python scripts/run_monthly_selection_m8_natural_industry_constraints.py --config config.yaml.backtest
```

各阶段职责：

| 阶段 | 入口 | 说明 |
| --- | --- | --- |
| M2 Dataset | `run_monthly_selection_dataset.py` | 构建月末截面、候选池、特征、未来一月 open-to-open 标签 |
| M3 Oracle | `run_monthly_selection_oracle.py` | 诊断候选池与标签上限 |
| M4 Baselines | `run_monthly_selection_baselines.py` | 线性、静态与树模型基线 |
| M5 Multisource | `run_monthly_selection_multisource.py` | PIT 基本面、资金流、股东户数等多源扩展 |
| M6 LTR | `run_monthly_selection_ltr.py` | 月度 learning-to-rank 模型 |
| M7 Report | `run_monthly_selection_report.py` | 生成研究版当月 Top-K 名单、贡献、行业暴露和完整证据 |
| M8 Regime/Industry | `run_monthly_selection_concentration_regime.py`、`run_monthly_selection_m8_natural_industry_constraints.py` | 状态切片、行业集中度治理与 promoted 默认方法证据 |

## 4. 候选池与可交易性

月度候选池只做准入过滤，不做 alpha 判断：

| 候选池 | 规则 |
| --- | --- |
| `U0_all_tradable` | 当期 OHLCV 有效，且下一交易日开盘非停牌、非一字涨停不可买 |
| `U1_liquid_tradable` | `U0` + 最小历史长度 + 20 日成交额门槛 |
| `U2_risk_sane` | `U1` + 剔除极端涨跌停路径、极端波动/换手和绝对高位标的 |

当前 active promoted 方法选择 `U1_liquid_tradable`，再通过 `M8_regime_aware_fixed_policy__indcap3` 做排序与行业只数硬约束。

## 5. 输出与证据链

月度脚本会写出 CSV、JSON manifest 和 Markdown 报告。常见输出包括：

| 输出 | 说明 |
| --- | --- |
| `*_recommendations.csv` | 月度 Top-K 名单 |
| `*_monthly_long.csv` | 历史月度长表，用于收益和切片复核 |
| `*_leaderboard.csv` | 模型与候选池组合的历史排序效果 |
| `*_rank_ic.csv` | 月度 Rank IC |
| `*_quantile_spread.csv` | 分桶 spread |
| `*_industry_exposure.csv` | 行业暴露 |
| `*_risk_summary.csv` | 风险摘要 |
| `*_manifest.json` | 研究身份、输入、参数和产物路径 |

研究报告默认归档到 `docs/reports/<YYYY-MM>/`，当前月度 promoted 证据见 `docs/reports/2026-05/monthly_selection_2026_05_top20_promoted_2026-04-30.md`。

## 6. Promotion 纪律

新增或替换默认月度方法前，至少需要：

- 正式 full backtest 和成本后表现。
- rolling / slice OOS 证据。
- strong-up、strong-down、宽基震荡等状态切片。
- Top-K 边界和候选池稳定性诊断。
- 行业集中度、换手、单票权重与可买性检查。
- `configs/promoted/promoted_registry.json` 新增记录，并由人工确认。

本系统输出的是研究与筛选结果，不构成投资建议。
